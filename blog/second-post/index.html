如何發佈個人網站到 Github Pages

1. 初次將個人網站發佈到 Github Pages

  • 創建 repository(資料夾)

  1. 打開 Github 首頁，點擊右上角的「+」號，點擊「New repository」創建一個新的 repository

  2. 輸入 Repository 的名稱

    • 命名規則：
      自己的 Githib 帳號.github.io

  3. 點擊下方綠色「Create repository」

  4. 點擊「Set up in Desktop」，來開啟 Github Desktop，幫我們把這個 repository clone(複製)到我們的電腦上

  5. 選擇自己想存放的位置

  6. 點擊「Clone」，將 repository clone 下來

  • 使用Hugo產生個人網站的程式碼

  1. 在桌面找到 exampleSite，在 exampleSite 上點擊右鍵，選擇開啟終端機

  2. 輸入「hugo」，按下 enter 鍵

  3. 個人網站的程式碼已生成！

  4. 將 exampleSite 中的「public」資料夾內所有檔案及資料夾複製，貼到我們 clone 的 repository (tsaipc2001.github.io) 裡面

  5. 回到 Github Desktop，(創建一個commit來儲存此次檔案改動)，在左邊「Summary」欄位打上commit訊息

  6. 點擊「Commit to main」

  7. 點擊右上方「Publish branch」，將 commit 上傳至 Github 上

  8. 回到Github頁面重新整理，即可看到 commit已成功上傳到 Github

  9. 開啟新分頁，輸入自己的 Github 帳號，按下 enter 鍵

  10. 約等待2-3分鐘，重新整理網頁，即可看到 Github Pages 成功將我們的個人網站程式碼轉換成精美的網站了！

—————————————————————————————————————————————-

2. 新增文章

  1. 在 VS code 上，exampleSite 的「content」下面的「blog」資料夾中創建一個檔案，自訂檔名，副檔名為「.md」

  2. 到之前寫好的文章中，複製 Hugo 的文章設定

  3. 修改標題、時間、圖片、description、分類、標籤

    • 修改圖片注意事項：

      先將要使用的圖片拉到「images」資料夾下面

  4. 以 Markdown 語法寫文章

  5. 到終端機上輸入「hugo」，按下 enter 鍵

  6. 將 exampleSite 中的「public」資料夾內所有檔案及資料夾複製，貼到我們 clone 的 repository 裡面(將 repository 中原有的內容全部取代)

  7. 回到 Github Desktop，在左邊「Summary」欄位打上 commit 訊息

  8. 點擊「Commit to main」

  9. 點擊右上方「Push origin」，將 commit 上傳至 Github 上

  10. 約等待2-3分鐘，重新整理網頁，就可在網頁上看到新增的文章了！
